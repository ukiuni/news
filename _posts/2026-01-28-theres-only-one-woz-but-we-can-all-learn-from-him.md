# There's only one Woz, but we can all learn from him - 「ウozは一人しかいない。でも誰もが彼から学べる」
魅力的なプロダクトは「自分の問題を解く」ことから生まれる──ウォズニアックに学ぶ、技術と人間性の結びつけ方

## 要約
Apple共同創業者スティーブ・ウォズニアックがTech Interactiveの「James C. Morgan Global Humanitarian Award」を受賞。受賞式で紹介された脳–機械インターフェース（BCI）や超音波ヘッドセット、パーキンソン症状を支援する装置などの事例を通じて、「自分の問題を解く」発想が生む技術と社会貢献の重要性を示した。

## この記事を読むべき理由
日本は高齢化や障害支援ニーズが増す中、BCIや補助デバイス、地域教育・カルチャーの重要性が高まっています。ウォズの実例は、技術開発がどう社会課題と結びつくかを学ぶ格好の教科書です。

## 詳細解説
- 受賞と場の背景：1月16日、サンノゼのTech Interactiveが主催するイベントでウォズが表彰。式では、障害者のコミュニケーションを助けるBCI（Cognixion等）、BCI利用者が倫理や実務を共に作るフォーラム、超音波を用いた精神医療ヘッドセット（Sanmai）、歩行を支援するデバイス（NexStride）などが紹介された。  
- 技術的ポイント：ウォズは1977年のApple IIで「色表示」を追加した実装を引き合いに出し、低コスト・低複雑性で“規則を破る”工夫が画期的だったと指摘。今回の受賞者もそれぞれ既存の枠を越え、実ユーザーの課題解決を起点に技術を設計している点が共通する。BCIはハード×ソフト×倫理の融合が必須で、ユーザー主導のコミュニティ（BCI Pioneers）の存在が技術受容性を高める。  
- 社会貢献の軸：ウォズは地元サンノゼの教育・文化施設（Tech Interactive、Children’s Discovery Museum）や学校のコンピュータ教室支援、初期のApple株の分配、EFFへの寄付など、技術者としての人的・資金的還元を長年続けている。技術的才能と同時に「教育・コミュニティへの投資」が彼の人道的評価を支えている。

## 実践ポイント
- 問題起点で作る：まず自分や対象ユーザーの具体的な困りごとを定義する。機能過剰より「最小実装で解決」を目指す。  
- ユーザーと共創する：特に医療・障害支援系は患者・利用者コミュニティを設計過程に入れる。倫理・運用ルールを共に作ること。  
- 低コストの工夫を探す：高価な部品に頼らず、既存資源の組合せで価値を出す発想を養う（ウォズの色表示のように）。  
- 地域投資と教育：プロダクトだけでなく、地域の教育・メイカースペース支援や若手育成にコミットすると長期的な信頼と人材が育つ。  
- 日本での応用例：高齢化対策のBCI応用、リハビリ支援デバイス、学校や地域のラボでの実践プロジェクトは特に実装価値が高い。

短く言えば、偉大な技術は「誰かの困りごとを解く」ことで生まれ、技術者の社会的貢献はプロダクトの社会的受容を大きく左右する――ウォズの足跡はそのシンプルな教訓を伝えています。
