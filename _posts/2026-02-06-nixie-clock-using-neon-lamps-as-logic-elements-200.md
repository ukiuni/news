# Nixie-clock using neon lamps as logic elements (2007) - ネオンランプを論理素子に使ったニキシー時計（2007年）

真空管とネオンランプだけで動く「シリコン無し」の自作ニキシー時計 — レトロ工作のロマンと技術的教訓

## 要約
作者はトランジスタやICを使わず、ネオンランプ＋抵抗・コンデンサ・ダイオードで論理（リングカウンタ）を組みニキシー管を駆動する時計を作った。部品特性（着火電圧と維持電圧）の違いや経年変化で設計は繊細になり、実用性確保には工夫と保守が必要だと結論づけている。

## この記事を読むべき理由
レトロ電子工作に興味がある人、あるいは「シリコンに頼らない回路」の原理を学びたい日本のエンジニア／メイカーにとって、ネオンランプを使った論理回路の実践的知見は貴重。日本は地域で50/60Hzが混在するため周波数由来の分周設計も参考になる。

## 詳細解説
- 構成の概略：交流線の周波数（作者は50Hz）をリングカウンタ群で段階的に分周し、1秒パルス→分→時のカウントを行う。ニキシー管の各カソードはネオンランプの光でLDR（光抵抗）を駆動して接地側に引き込み表示する仕組み。
- ネオンランプを論理素子にする仕組み：ネオンは「着火電圧（striking）」が高く、「維持電圧（maintaining）」が低いというヒステリシスを利用し、1つのランプでパルスを伝搬させるリングカウンタが作れる。ただし現行の市販ネオンは着火／維持差が小さく、古典的な部品ほど安定しない。
- 実設計上の工夫と問題点：
  - 市販ネオンのばらつき対策としてランプ特性を測定・ラベル管理し、抵抗値を調整してマッチングする必要がある。
  - ランプの初期数時間で特性が変化するため「バーンイン（エージング）」が必須。
  - リング同士を直接カスケードできない場合があり、増幅用にバッファ用ネオンを追加してわずかなパルスで打ち上がるようにした。
  - 光漏れや周囲光に敏感なため、LDR周りに遮光・フィルタ（黒ディスク＋赤フィルム）を入れ、場合によっては低輝度青LEDで環境光を与える対策を実施。
  - 長期的にはランプの経年変化で安定性が低下し、最終的に実用性を失った経験がある（作者は後に別ランプで2020年に改良版を制作）。
- 電源と安全：ニキシー／ネオン回路は高電圧を扱うため給電回路のきれいな設計と透明な筐体での絶縁・安全対策が重要。作者は150B2のような安定管を電源に使う改良も行っている。

## 実践ポイント
- まず自分の地域の商用周波数（50Hz/60Hz）を確認して分周設計を合わせる。
- ネオンランプは個体差が大きいので、着火電圧／維持電圧を測りラベル管理してから回路に組み込む。バーンインを必ず行う。
- リングカウンタを確実に動かしたければ、各段にバッファ用ネオンを追加する（小さな信号で打ち上がるようにバイアス）。
- ニキシー駆動にLDRを使う場合、周囲光の影響を防ぐ遮光と小径の光導入で安定表示を得る。実験用には青LEDなどで弱照明を与えると有効。
- 長期運用を目指すなら、ネオンのみの構成は維持が大変なので電源の安定化（安定管）や、駆動を一部マイコン／トランジスタで補うハイブリッド化を検討する。
- 高電圧作業は感電リスクがあるため、絶縁・放電処理・保護カバーを徹底する。

最後に：古典的な電子回路の原理を現物で体験できる魅力的な工作だが、「ロマン」と「現実的な保守性」のトレードオフを理解して始めるのが成功の鍵。
