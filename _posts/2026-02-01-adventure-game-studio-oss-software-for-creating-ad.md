# Adventure Game Studio: OSS software for creating adventure games - アドベンチャーゲーム制作向けオープンソース「Adventure Game Studio」
レトロな良作を自分で作る！無料＆オープンソースの「Adventure Game Studio」始め方ガイド

## 要約
Adventure Game Studio（AGS）は、グラフィカルなポイント＆クリック型アドベンチャーゲームを手軽に作れる無料のオープンソースIDEで、Windowsベースの統合ツールで制作→テスト→公開まで進められます。

## この記事を読むべき理由
日本でもノベル系や2Dアドベンチャーは根強い人気。AGSはプログラミング初心者でも取り組みやすく、海外の豊富なコミュニティ資産やジャム（MAGS）を活用すれば短期間で作品公開まで持っていけます。

## 詳細解説
- 開発環境：Windows向けのIDEに、グラフィック取り込み、スクリプト編集、テスト実行が統合されており、手順が分かりやすい。最新版はAGS 3.6.2（Patch 6）として配布されています。  
- スクリプトとエンジン：AGSは独自のスクリプト（Cライクな文法）で動作ロジックや会話、インベントリ操作、ホットスポット検出などを記述します。ルーム（マップ）単位での管理やカットシーン制御も可能です。  
- マルチプラットフォーム再生：作成したゲームはWindows以外にLinux、iOS、Androidなどで再生可能な対応がされており、配布の幅が広い点が強みです。  
- コミュニティ：公式フォーラム、Discord、MAGS（月例のゲームジャム）やAGSアワードなど、学習・フィードバック・公開の場が充実。海外資産の利用や英語ドキュメントを活かして学べます。  
- 収益化・公開：無料で作ってサイトにアップロード可能。商用作品も存在し、インディー展開の選択肢が広いです。

## 実践ポイント
1. 公式サイトからAGS 3.6.2（Patch 6）をダウンロードしてIDEに触る。  
2. まずは既存のサンプル／短編ゲームをプレイして、操作感とシステム構成を理解する。  
3. 小さなワンルームの短編を作り、スクリプトで会話とアイテム処理を実装してみる。  
4. フォーラムやDiscordで疑問を投げ、MAGSなどのジャムで公開とフィードバックを得る。  
5. モバイルやLinux向け配布を想定するなら、解像度や入力（タッチ操作）を早めに検証する。

短時間でプロトタイプを作りやすく、国内の同人／インディー創作とも親和性が高いツールです。まずは一度触って小さな作品を完成させてみましょう。
