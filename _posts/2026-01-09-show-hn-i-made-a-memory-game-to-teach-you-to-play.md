# Show HN: I made a memory game to teach you to play piano by ear - ピアノを耳で弾く練習ができる記憶ゲームを作りました
「遊びながら『耳で弾く力』を鍛える」——短時間で続けられる音感トレーニングの新提案

## 要約
短いメロディを記憶して再現する“記憶ゲーム”を通じて、ピアノを耳で弾く能力（耳コピ／相対音感）を鍛えるウェブアプリの紹介。ゲーム感覚で継続しやすく、ブラウザで手軽に試せる点が特徴です。

## この記事を読むべき理由
- 楽器の初心者や子どもが「音を聞き分ける力」を楽しく身につけられる方法を知りたい人向け。  
- 日本では個人レッスンや幼児教育での音感トレーニング需要が高く、授業や自習ツールとして活用できる可能性があるため参考になるからです。

## 詳細解説
元記事（および公開サイト）では、プレイヤーに短い音列（メロディ）を聞かせ、それを覚えて再現させるというゲーム性を採用しています。ゲームの狙いは単なる記憶力ではなく、「音程の違い」「旋律の形」「リズム」を耳で判別して手で再現する訓練にあります。

想定される技術構成（ウェブアプリとして実装されている場合）
- 音声合成／出力: Web Audio API や Tone.js といったライブラリでピアノ音を生成。  
- UI: キーボード（画面上の鍵盤）をクリック／タップして入力、あるいはMIDIキーボードを接続して入力できる設計もあり得る。  
- ゲームロジック: シーケンス生成・段階的難易度（長さ・音域の拡大・リズムの複雑化）・スコアリング。  
- 永続化: localStorage やサーバーを使ったスコア保存、学習履歴の管理（続けやすさ重視）。  
- レスポンシブ対応: スマホ・タブレットでも遊べるように設計されている可能性が高い。

教育的なポイントとしては、絶対音感ではなく相対音感（音の関係性）を重視する点が優れています。相対音感は実践的な耳コピや伴奏力につながり、短い反復セッションの積み重ねで上達が見えやすいです。

## 実践ポイント
- まずはヘッドホンで短いセッション（5〜10分）を毎日行う。継続が最も効果的。  
- テンポを落として音の違いを意識し、慣れてきたら徐々に速くする。  
- スクリーン鍵盤だけでなく、持っているキーボード／デジピ（MIDI）で入力できるなら接続して実践的に。  
- 開発者向け：簡単に同様のツールを作るなら Tone.js（音声）＋小さな状態機械でシーケンス管理、localStorageで学習進捗を保存すると短時間でプロトタイプが作れます。

参考に、ブラウザでピアノ音を鳴らす最小例（Tone.js）：
```javascript
// javascript
import * as Tone from "tone";
const synth = new Tone.Synth().toDestination();
await Tone.start();
synth.triggerAttackRelease("C4", "8n");
```
