# Windows 11 Notepad to support Markdown - Windows 11 のメモ帳が Markdown に対応
メモ帳がMarkdown＋AIで大進化。書く・要約する・描くがもっと速く、もっと直感的に。

## 要約
Windows 11のNotepadとPaintにアップデートが配信開始（Insider Canary/Dev）。NotepadはMarkdownの軽い書式（取り消し線・入れ子リスト等）とAI出力のストリーミング表示を追加。Paintはテキストプロンプトから塗り絵を生成する「Coloring book」と塗りつぶしの許容度スライダーを追加。

## この記事を読むべき理由
日常的なメモ・ドキュメント作成や教育コンテンツ作り、軽い画像編集がより手早くできるようになり、日本の開発者・教育現場・クリエイターにも実務的な恩恵があるため。

## 詳細解説
- Notepad（ver. 11.2512.10.0）
  - Markdownの軽量フォーマット拡張：取り消し線（strikethrough）やネストしたリストがツールバー／キーボードショートカット／Markdown直接編集で利用可能。
  - 初回表示の「What’s New」ウィンドウを追加：新機能の発見をサポート（右上のメガホンアイコンで再表示可）。
  - AIテキスト機能（Write/Rewrite/Summarize）でストリーミング結果を導入：ローカル／クラウド生成問わず、応答のプレビューが早く表示される。利用にはMicrosoftアカウントでのサインインが必要。
- Paint（ver. 11.2512.191.0）
  - Coloring book：Copilotメニューからテキストプロンプトを入力して塗り絵ページをAI生成（「a cute fluffy cat on a donut」など）。生成物はキャンバスに追加・コピー・保存可能。Copilot+ PCでの利用制限あり。サインイン要。
  - 塗りつぶしの許容度スライダー：Fillツールの精度を割合で調整し、きれいな塗りつぶしや表現の幅をコントロール可能。
- フィードバックはFeedback Hub（WIN + F）の Apps > Notepad / Paint に送信。

## 実践ポイント
- まずInsiderのCanaryまたはDevチャネルで該当バージョンに更新する。  
- NotepadでMarkdownを試す：ツールバー／Ctrl系ショートカットで取り消し線や入れ子リストを使ってみる。  
- AI機能はMicrosoftアカウントでサインインし、Write/Rewrite/Summarizeでストリーミング結果の応答を体験。  
- PaintのColoring bookはCopilotメニュー→プロンプト入力→Generate→Add to canvasで即利用（Copilot+ PCの対応状況を確認）。  
- 不具合や要望はWIN + FでFeedback Hubへ投稿して製品改善に協力する。
