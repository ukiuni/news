# Show HN: Artificial Ivy in the Browser - ブラウザ上の人工アイビー
魅せる“生態系”を自分で育てる──マウスひとつで増殖する人工つたのライブシミュレーション

## 要約
ブラウザ上で動く「人工アイビー」シミュレーション。単純な確率ルール（成長／分裂・向きの変化・寿命・信号デカイ）を組み合わせるだけで、複雑で有機的な蔓状パターンが自律的に生成される。

## この記事を読むべき理由
- ビジュアルが魅力的で、ジェネレーティブアートやゲームのプロトタイプに即使える。  
- シンプルなルールから複雑な挙動が生まれる「創発（emergence）」の教材として、学習やワークショップに最適。  
- 日本のデジタルアート展示、教育、インディーゲーム開発などで応用しやすい。

## 詳細解説
シミュレーションの基本要素と挙動は以下のとおりです。

- 初期状態は1つのセル（種）。各フレームでセルは「成長するか」「分裂するか」を確率で決定します。ユーザーはスライダーで確率を調整可能（Grow chance, Split chance）。
- 分裂（split）したセルは元の方向からわずかに角度を変えた方向に新しいセルを作ります。最大の曲がる角度（Max turn angle）を小さくすると直線的に伸び、大きくすると枝分かれが多くなります。
- 成長・分裂できるのは「若い」セルだけ。年齢の上限はその瞬間のセル年齢分布のパーセンタイルで決まります（Age threshold %ile）。閾値を上げるとより多くのセルが活動可能になります。
- セルは生成位置に“信号”を残し、他のセルが同じ場所で成長・分裂するのを抑えます（重なり防止）。信号は毎フレーム減衰し、減衰率（Signal decay rate）を上げると早く消えるため再成長が起きやすくなりますが、重なりも起きやすくなります。
- セルは最大寿命に達すると死にます。

重要な数式（概念的）:
- 信号の減衰（1フレーム毎）を簡単に表すと  
  $S_{t+1} = S_t \times (1 - r)$  
  ここで $r$ はデカイ率（0〜1）。
- 年齢閾値は分布のパーセンタイルなので、例えば $p$ パーセンタイルなら「年齢を昇順に並べたとき上位から $p\%$ に当たる値」が閾値になります。

簡単な処理ループ（概念）:
```javascript
// javascript
cells.forEach(cell => {
  cell.age++;
  if (cell.age < ageThreshold && random() < growProb) grow(cell);
  if (cell.age < ageThreshold && random() < splitProb) split(cell);
});
signals.forEach(s => s.value *= (1 - decayRate));
removeDeadCells();
```

## 実践ポイント
- 見た目の作り方:
  - 直線的な「つた」を作りたい → Max turn angle を小さく、splitProb を低めに。
  - モコモコした茂みを作りたい → splitProb と growProb を高め、turn angle を大きく。
  - 再成長・オーバーラップを活かした動的な表現 → signal decay を速く（高め）に設定。

- 実装ヒント（Web/教育向け）:
  - 描画は Canvas か WebGL で。多数セルなら描画バッチ化／オフスクリーンを検討。  
  - 信号はグリッド（2D配列）か空間ハッシュで扱うと高速。  
  - 年齢のパーセンタイル計算は中央値計算と同様にソートして索引を取ればOK（頻繁ならヒープや選択アルゴリズムを利用）。

- 活用アイデア:
  - 教室：確率や生成アルゴリズムの直感的デモ。  
  - 作品：インタラクティブインスタレーション、背景生成、プロシージャルテクスチャ。  
  - プロトタイプ：ゲームの植物成長や資源分布の可視化。

このシンプルなルールセットは少しのパラメータ調整で多様な表現を生むので、クリエイティブな実験や教育用途に最適です。リンク先で実際にスライダーをいじりながら「育てる」体験をしてみてください。
