# Show HN: I built a space travel calculator using Vanilla JavaScript - バニラJSで作った「宇宙移動距離計算機」
あなたは生まれてから地球ごと宇宙をどれだけ移動したか知っていますか？バニラJavaScriptで動く「Cosmic Odometer」を覗いてみましょう。

## 要約
生年月日を入力すると、地球の自転・太陽周回・太陽系の銀河回転・銀河の宇宙運動を合算して、あなたが生まれてから移動した“総距離”や月往復換算・光年進捗などをリアルタイムで表示するクライアントサイドのツールです。

## この記事を読むべき理由
- 身近な生年月日から宇宙規模の距離感を直感的に理解できる教育的ツールで、学習教材やデモに最適。  
- 実装は純粋なVanilla JSで、ブラウザだけで完結するため日本の開発者が学び・再利用しやすい。  
- プライバシー重視（生年月日はクライアントだけで処理）で、個人情報に敏感な日本市場でも受け入れやすい設計。

## 詳細解説
基本原理はシンプルです。時間差を計算して、それぞれの速度に掛け合わせるだけです。
- 使う物理式: $d = v \times t$（距離 = 速度 × 時間）  
- 計算する速度の内訳（サイト表記に基づく概算）:  
  - 地球の自転（緯度依存、赤道付近で約1,600 km/h ≒ 0.465 km/s）  
  - 地球の公転（太陽周回、約107,000 km/h ≒ 29.7 km/s）  
  - 太陽系の銀河回転（約792,000 km/h ≒ 220 km/s）  
  - 銀河自体の運動（宇宙背景に対する相対速度、約2,100,000 km/h ≒ 583 km/s）  
- 合計距離はそれぞれの経路長を単純合算した“旅した総距離”。「直線変位」は円軌道や自転を無視した場合の概算的な純粋な位置差を示します（ベクトル合成や軌道位相を完全に再現するわけではありません）。  
- 実装上のポイント: ブラウザのDateオブジェクトで経過時間をミリ秒単位で取得し、秒→時間へ変換して速度と掛け合わせる。表示は単位切替（km/miles）やリアルタイム更新、コピーリンクなどのUI機能を備える。  
- 注意点: 速度は定数近似（時間変化や相対論的効果は無視）なので「厳密な天文軌道計算」ではなく、視覚的・教育的な概算ツールです。プライバシーはサイト側で明示されている通り完全クライアント処理。

## 実践ポイント
- まずサイトを開いて自分の生年月日で試す（クライアント処理で安全）。  
- ソースを読むとVanilla JSでの時間差計算・リアルタイムUIの作り方が学べるので、授業用デモや社内ハッカソン題材に最適。  
- 日本語化・表示単位や速度の精度改善（緯度での自転速度計算や軌道位相の導入）をローカルで拡張してみると良い学習になる。  
- 学校や子ども向けには「距離を月往復換算」「光年進捗」の表示が直感的でウケるので、解説スライドに組み込むと効果的。
