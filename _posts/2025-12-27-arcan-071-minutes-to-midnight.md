# Arcan 0.7.1 — Waylandゲームから“量子耐性”ネットワークまで、デスクトップ環境を再発明する小さな大型アップデート

## 要約
Arcan 0.7.1 がリリース。Wayland上でのゲーム互換性改善、再接続（connection resumption）とキャスト機能、そして前方秘匿性を強化するポスト量子暗号（ML‑KEM）の導入など、ネットワークとランタイムの実用機能が中心。

## この記事を読むべき理由
ローカル/リモートのウィンドウ管理やゲーム配信、セキュアなリモート接続に興味がある日本の開発者・パッケージ保守者にとって、Arcanは“軽量かつ柔軟”な代替UI基盤として有望。Wayland周りや配布ルート（Fossil/Codeberg）への対応方針は国内ディストリビューションや組込み用途にも影響する。

## 詳細解説
- リリースの背景  
  - 2025年末にタグ付けされた安定寄りのリリース。主要開発はFossilへ移行、CodebergへGitミラーを用意しているため、パッケージ配布はCodeberg/Fossilを参照することが推奨される。  
  - 開発コミュニティではメンバーの訃報があり、0.8ブランチの性能工学トピックはそのメンバーに捧げられる予定。

- ネットワーク／セキュリティ周りの主な改善  
  - ML‑KEM（多線形鍵包絡のポスト量子方式）をラチェット型鍵更新に組み込み、将来的な「収集して後で解読」攻撃に対する備えを強化。ファイル転送に対する署名検証キーの導入や、検索要求のロード分散に使うPoW（Proof‑of‑Work）も追加。  
  - 接続再開（connection resumption）をサポート：クライアントが切断されてもソース側のアプリケーション状態を保持し、再接続時にそのまま再ペアリングできる。遠隔アプリの中断耐性が向上。  
  - キャスト機能（--cast）を追加：最初に接続したユーザーが「ドライバー」になり、それ以降の接続は読み取り専用のストリームを受け取るモードを提供。

- ディレクトリとリンク設計  
  - エンドポイントは Source / Sink / Directory のいずれかの役割を持つ設計を継続。ディレクトリサーバ周りの改良で、ネットワーク上のリソース探索や参照の扱いが整理されている（統一的かつ参照可能なリンク運用へ）。

- ハックイベント成果と周辺ツール  
  - Gamescope のポートにより、SteamをXwayland上で動かす際の互換性問題を軽減（Baldur's Gate 3 の動作例あり）。  
  - Qt5/Qt6向けプラットフォームプラグインの改善で Qbittorrent や Binary Ninja 等は動作するが、FreeCAD のような2D/3D混在ウィンドウにはまだ課題あり。  
  - Xkbd2Lua により X のキーボードレイアウトを静的に変換、libxkbcommon への依存を削れるオプションを提供。  
  - Atro の「Lasso」など実験的なインタラクティブキャンバス型ウィンドウマネージャや、A12プロトコル用のポータブルビューア作業などが進行中。  
  - 静的ビルドで Arcan + Durden + Cat9 をブート可能にする取り組みがあり、nix のワンライナーでビルドする手順が公開されている（リリースノート参照）。

- パッケージャ／運用者向け注意事項  
  - GitHubのリポジトリはミラーであり、以降のミラーは更新されない可能性があるため、公式な開発履歴／パッケージは Fossil または Codeberg を参照すること。

- 小話（運用IDなど）  
  - DECT拡張が6681、かつ過去に使われたコントロールポート番号と同一というエピソードあり（プロジェクトらしい遊び心）。

## 実践ポイント
- すぐ試す
  - Codeberg のミラーから最新タグを取得して試す（パッケージ保守者は Fossil/Codeberg をソースに切り替える）。  
  - nix ユーザーは公開されている nix-flake ビルドを試して、静的イメージ／テスト環境を素早く構築する。  
- 運用で注目すべき点
  - ネットワーク運用者は ML‑KEM 対応やラチェットの挙動を検証し、「収集して後で解読される」リスクに備える。  
  - リモートデスクトップ／配信用途では connection resumption と --cast を組み合わせ、切断耐性と閲覧権限の分離を検討する。  
- パッケージング／軽量化
  - Xkbd2Lua を利用して libxkbcommon の依存削減を検討。組込みや最小限イメージでの利点がある。  
- 開発者向け
  - Gamescope ポートや Qt プラグインの改善点を確認し、Wayland 化やゲーム動作検証の参考にする。
