# Some Silly Z3 Scripts I Wrote - いくつかのくだらないZ3スクリプト
たった数行で「方程式の解」も「乱数の逆解析」もできる！Z3で遊ぶ実践ガイド

## 要約
Z3はSMT（Satisfiability Modulo Theories）ソルバーで、制約を与えると変数の割当（モデル）を自動で見つける。この記事では入門向けに、簡単な連立方程式、数の組み合わせ探索、最適化、擬似乱数の逆解析、定理証明、株の売買最適化といった実例をわかりやすく紹介する。

## この記事を読むべき理由
国内でもテスト自動化、形式手法、逆解析、探索的バグ発見などSMTの実用性が高まっている。Pythonバインディングで手軽に始められるので、エンジニアが即座に試せるツール知識として有益。

## 詳細解説
何ができるか（簡単な定義）
- Z3は「制約（方程式や不等式）」を受け取り、その制約を満たす変数の値を探す。満たせれば "sat"、なければ "unsat" を返す。  
- SMT は数学（整数、実数）、配列、正規表現、量化子などの理論を組み合わせて扱える点が強み。

基本的な使い方（例）
- 連立方程式 $4x + 2y = 14$ と $2x - y = 1$ を解く簡単な流れ：

```python
from z3 import *
s = Solver()
x, y = Ints('x y')
s.add(4*x + 2*y == 14)
s.add(2*x - y == 1)
print(s.check())
if s.check() == sat:
    print(s.model())
```

- 異なる数同士で和と積が等しい組があるか（Distinct, Product）：
  - Distinct() や IntVector を使うと同値条件や多変数の扱いが楽になる。

最適化（Optimize）例：年金積立風の最小入金額
- 口座に年1%で、毎年同額を入れたとき20年後に1万になる最小入金額をOptimizeで求める。Real 型と再帰的な残高式を使うことで表現する。

乱数（LCG）の逆解析
- 線形合同法（LCG） $X_{n+1} = (a X_n + c) \bmod m$ の既知出力列から $a, c$ を求めることができる。実務では古いPRNGの解析やリバースエンジニアリングに有用。

定理証明としての利用
- 命題の否定（反例）を探すことで定理の妥当性をチェック。たとえば加法の可換性が反例を取れないか調べると「Theorem true」となる。

配列（Array）とインデックスの罠
- Z3の配列は「全てのインデックスに値を返す全域関数」扱い。Pythonのリストと混同すると、インデックス制約を付け忘れて意図しないモデル（巨大なインデックス等）を得る。配列を使う場合は必ず範囲（0 ≤ i < n）を明示する。

株の売買最適化（配列利用＋Optimize）
- 値の入った配列をZ3の Array として定義し、買いと売りのインデックスを最適化する。インデックスに範囲制約を与える点が重要。

## 実践ポイント
- セットアップ: pip install z3-solver（Pythonバインディング）。
- 小さく始める：まずは Int, Real, Solver() の簡単な例でsat/unsatを体感する。
- 配列はPythonリストとは別物：Select／Store を理解し、インデックスの範囲制約を必ず追加する。
- Optimize は便利だが重い：数値最適化用途なら専用ソルバー（MiniZincなど）も検討する。
- 応用領域：形式検証／テスト入力生成／簡易逆解析／探索問題のプロトタイピングに向く。
- 学習リソース：Hillel Wayne の記事や「Logic for Programmers」など実例が多い資料を参照すると習得が早い。
