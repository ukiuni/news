# Show HN: Zap Code – AI code generator that teaches kids real HTML/CSS/JS - Zap Code：子どもが本物のHTML/CSS/JSでアプリを作るAIツール
子どもでも“言葉で指示→即アプリ”ができる！遊びながら本当のコードを学べる新サービス

## 要約
英語でアイデアを入力するとAIが実際に動くHTML/CSS/JavaScriptを生成する学習プラットフォーム。視覚編集→コードの「のぞき見」→直接編集の3モードで8〜16歳向けに段階的に学べる。

## この記事を読むべき理由
日本でも小中高校・プログラミング教室で「実用的なWebスキル」を早期に身につけたいニーズが高まっているため、子ども向けに本物のコード体験を与える点で注目に値する。

## 詳細解説
- 入力：自然言語（英語）で「こういうゲームを作って」と説明するとAIが即座にWebアプリを生成。ライブプレビューで動作確認可能。  
- 生成物：出力は実際のHTML、CSS、JavaScript。ブロック式ではなく“テキストコード”なので学習した知識がそのまま転用できる。  
- 学習モード：  
  - Visual（視覚的に見た目を操作）  
  - Peek（コードを覗いて学ぶ）  
  - Edit（直接コードを書いて改造）  
  これにより初心者〜経験者まで段階的に深掘り可能。  
- 共有とコミュニティ：作品はギャラリーに公開でき、他者のプロジェクトをプレイ／リミックスして学べる。  
- 安全性と運用：親用ダッシュボード、作品モデレーション、広告・トラッキング無しを明記。無料で始められるフリーミアム設計。  
- 教育的価値：即時フィードバック＋実コード生成で「結果が見える学習」が可能。実務で使う言語（HTML/CSS/JS）なので将来の学習コースや就業にもつながる。

## 実践ポイント
- 最初のプロンプト例：「Make a quiz app about Japanese prefectures」など具体例で試す。  
- 教室導入：ワークショップで「Visual→Peek→Edit」の順に課題を出すと効果的。  
- 保護者向け：ダッシュボードで制作物と時間を監視し、ポートフォリオ作成を促す。  
- 学習法：生成されたコードをコピペしてローカルで実験、差分を作って挙動を比較する。  
- コミュニティ活用：他の作品をリミックスしてアイデアと実装パターンを学ぶ。

興味があれば、まずは短い英語の説明文で1プロジェクト作ってみると、このサービスの学習フローが体感できます。
