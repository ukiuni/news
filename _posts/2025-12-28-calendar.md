# Calendar - カレンダー

## 要約
Neatnikの「Calendar」は、年の全日付を1ページに自動フィットで出力するシンプルな印刷カレンダー。用紙サイズや向きを気にせず瞬時に年間プランを俯瞰できます。

## この記事を読むべき理由
日本では紙の手帳や社内の「年間一覧」を壁やデスクに貼って使う文化が根強く、A4一枚で全体像を把握できるツールは実務でも個人の時間管理でも非常に実用的です。2026年の年間計画や会議・リリースのマイルストーン可視化にすぐ使えます。

## 詳細解説
- 基本機能  
  ページはURLパラメータ（例: ?year=2026）で年を指定でき、HTML/CSSで全日付を1ページに並べる構成になっています。見た目は最小限で、印刷時に余白を詰めればどんな紙サイズでも年全体が収まるよう調整されています。

- 技術的ポイント（推測と再現のヒント）  
  - 印刷最適化: CSSの @media print と @page 指定でランドスケープ推奨・余白最小化を指示。ブラウザ印刷時にヘッダー／フッターをオフにすると余白を完全に使えます。  
  - レスポンシブレイアウト: CSS Gridやテーブル風レイアウトで月ごとのセルを均等に配置し、フォントや行間を縮めて全体を詰める。ブラウザ側で自動スケーリングさせる実装が多いです。  
  - 動的生成: JSで年の開始曜日や月の日数を計算してDOMに出力する仕組み。URLの year パラメータを読み取って表示年を変えられます。

- 実用性の理由  
  1ページに年間を並べると、四半期やプロジェクトの切れ目、連休・祝日の連なりをビジュアルで素早く把握できます。物理で持ち歩けるためふとした瞬間に予定を書き込みやすい利点があります。

## 実践ポイント
1. 印刷設定  
   - 用紙向き: 横（landscape）推奨。  
   - ヘッダー/フッター: ブラウザの印刷設定で無効化（ページ上部の余白削減）。  
   - マージン: 最小またはなしに設定してA4/B5にフィットさせる。  
2. PDF化して配布  
   - ブラウザで「PDFに保存」して社内配布。A4で配ればそのまま共有資料になります。  
3. 日本向けカスタマイズ案  
   - 祝日表示: holiday-jp 等のライブラリ/APIで国民の祝日を塗りつぶす。  
   - 会計年度マーカー: 4月を起点に色付けして新年度計画を視覚化。  
4. 開発者向け再現スニペット（例）  
   ```javascript
   // 例: URLパラメータから年を取得してカレンダー生成
   const params = new URLSearchParams(location.search);
   const year = parseInt(params.get('year') || new Date().getFullYear(), 10);
   // 年の各月の日数と開始曜日を計算してDOMにレンダリングするロジックへ
   ```
   ```css
   /* 印刷向けの簡単なヒント */
   @media print {
     @page { size: landscape; margin: 0; }
     body { font-size: 8pt; line-height: 1; }
   }
   ```
