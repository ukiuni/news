# The Unix Pipe Card Game - Unixパイプカードゲーム
魅せる「パイプ」で学ぶ！カードで遊びながら身につくUnixコマンドの直感

## 要約
カードゲーム形式で子どもや初心者にUNIXのパイプ（|）を使ったコマンド連結を教える教材。基本コマンドを組み合わせて課題を達成することで、実践的なスキルが身につきます。

## この記事を読むべき理由
サーバ操作やデータ処理で必須のシェル操作は、学習の敷居が高く感じられがちです。カードゲームに落とし込むことで楽しみながら「パイプで繋ぐ思考」が身につき、国内の教育現場や入門ワークショップですぐ使えます。

## 詳細解説
元記事のゲームは、cat, grep, tail, head, wc, sort, uniq といった基本コマンドを前提に、タスクカードに書かれた「目的」を達成するために手持ちのコマンドカードでパイプチェーンを作るというもの。例えば「ファイル中で最も出現頻度の高い行を表示する」課題では次のようなチェーンを作ります。

```bash
cat 03.txt | sort | uniq -c | sort -n | tail -1
```

ここでのポイントは、
- catで内容を取り出す
- sortで同じ行を隣接させる
- uniq -cで出現回数を数える
- sort -nで数値で並べ替え
- tail -1で最頻出行を取る
という処理分解の考え方です。ルールは年少者がタスク形式（最短チェーン勝ち／最長チェーン勝ち）を選ぶなど簡単なラウンド制。拡張パックではプロセス代替（process substitution）や paste/tr/cut/bc といった追加ツールを学べます。作者はカードのPDF（印刷用）や箱のテンプレも公開しており、CC BY 4.0で配布されています。

## 実践ポイント
- 初心者向けの順序: まずcat/grep/sort/wc/head/tailを画面で実演してからカードで遊ぶと理解が早い。
- 教室導入: 3〜6人でラウンドを回せば協働学習に最適。最短チェーンルールは効率的思考を促す。
- ローカライズ案: 日本語テキスト（例：ログや短い文章）でタスクを作ればより実務寄りに。
- 環境がない場合: jslinuxなどブラウザ上のUNIX環境で動作を確認できる。
- 発展学習: 拡張パックでプロセス代替やcut/trを加え、テキスト処理パイプラインの幅を広げる。
