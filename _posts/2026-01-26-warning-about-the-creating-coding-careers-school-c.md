# Warning About  The Creating Coding Careers School (CCC) Scam - Creating Coding Careers School（CCC）詐欺の警告
あなたのキャリアを奪うかもしれない申し込み前に確認すべき5つのポイント：CCCに関する投稿から学ぶ安全確認術

## 要約
LinkedIn上の警告投稿をきっかけに、CCC（Creating Coding Careers）というスクールについて不審な点が指摘されています。本記事は「真偽未確認の警告」を紹介しつつ、同種のスクール被害を避けるための実務的な確認手順を整理します。

## この記事を読むべき理由
日本でも転職目的で短期スクールを利用する人が増えています。誇大な就職保証や曖昧な契約は金銭的／時間的損失につながるため、事前に確認すべきポイントを知っておくことが重要です。

## 詳細解説
- 出所の扱い：今回の情報源はLinkedInの個人投稿。個人の投稿は有益でも感情的な表現や誤情報が混ざる可能性があるため、単独で結論を出さない。
- よくある詐欺パターン：
  - 「短期間で高額内定」「返金保証」の曖昧な条件表示。
  - 架空の卒業生や捏造された就職実績（写真・コメントの使い回し）。
  - 契約書が不完全、重要事項が口頭だけ。
  - 支払いを個人振込や暗号通貨で要求。
- 技術的に確認できる点：
  - カリキュラムの具体性：使用する言語・フレームワーク、実プロジェクトの有無、メンター比率を確認。
  - ポートフォリオの検証：卒業生のGitHubのコミット履歴、プルリク履歴、公開プロジェクトの実態をチェック。短期間で不自然な大量のコミットやコピペは要注意。
  - データの整合性：求人提携企業名、就職率の算出方法（母数・期間）を明記しているか。
  - 資格・認証の真偽：表示している認定や証明書の発行元を公式サイトで確認。
- 法的・消費者保護面：
  - 日本では訪問販売や通信教育のクーリングオフ、消費者センターへの相談窓口がある。海外運営や日本未登録の法人は対処が難しい可能性がある。

## 実践ポイント
1. 情報源を多角的に検証する（公式サイト、法人登記、第三者レビュー、SNSの履歴）。
2. 契約前に書面で「返金条件」「就職保証の定義」「仲介先企業リスト」を要求する。
3. 支払いはクレジットカード等の追跡可能な手段を使い、個人振込は避ける。
4. 卒業生のポートフォリオ（GitHub等）を実際に確認し、コードの質と貢献履歴をチェックする。
5. 不安があれば消費者センターや労働相談窓口、弁護士に相談する（日本国内での被害なら自治体窓口が有効）。

（出典）議論の発端はLinkedInの投稿で、記事はその投稿を起点に「検証方法」と「予防策」を整理したものであり、CCC自体の有罪・無罪を断定するものではありません。
